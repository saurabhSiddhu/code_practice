## 📖 **HOW TO USE THIS TRACKER**

### ✅ **Daily Checklist Format**

- **Time Block**: Exact time to start each task (matches daily schedule)
- **Problem**: Specific LeetCode/Educative problem name
- **Target Time**: Maximum time to spend (move on if exceeded)
- **Success Criteria**: What constitutes completion
- **Documentation**: Required notes/templates to create

### 🎯 **Weekly Success Gates**

Each week has **MANDATORY** requirements you must pass before advancing:

- **Problem Count**: Minimum problems solved with accuracy threshold
- **Speed Targets**: Maximum time limits for Easy/Medium/Hard problems
- **Typing Speed**: WPM target with algorithm syntax
- **Pattern Mastery**: Recognition speed and implementation fluency

### 📊 **Progress Tracking**

- ✅ **Green**: Completed successfully
- 🟡 **Yellow**: Partially completed (needs review)
- ❌ **Red**: Failed (requires retry)
- ⏭️ **Skip**: Advanced past (mark reason)
