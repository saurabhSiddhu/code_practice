# 🛠️ Productivity Tools Setup Guide

## 📱 **Essential Apps & Tools**

### **Time Tracking & Focus**

1. **Toggl Track** - Time tracking for work blocks

   - Set up projects: "Work Tasks", "Interview Prep", "Exercise"
   - Use for pomodoro timing and productivity analysis

2. **Forest App** - Focus sessions with gamification

   - Set 90-minute focus blocks
   - Track daily focus streaks

3. **Be Focused** - Pomodoro timer
   - 25-minute work sessions with 5-minute breaks
   - Customizable for 90-minute ultradian rhythm blocks

### **Interview Preparation**

1. **LeetCode Premium** - Algorithm practice

   - Daily coding challenges
   - Company-specific problem sets

2. **Anki** - Spaced repetition flashcards

   - Create decks for system design concepts
   - Data structures and algorithms review

3. **Excalidraw** - System design diagrams
   - Practice drawing architecture diagrams
   - Whiteboarding preparation

### **Health & Wellness**

1. **Apple Health / Google Fit** - Exercise tracking

   - Monitor heart rate during workouts
   - Track sleep quality and duration

2. **Headspace / Calm** - Meditation
   - 10-minute daily meditation sessions
   - Stress management techniques

## ⚙️ **VS Code Snippets Installation**

1. Open VS Code
2. Go to **File > Preferences > Configure User Snippets**
3. Select **vue.json** or create new global snippets file
4. Copy the snippets from `CODING_PATTERNS.md` into the file

## 📱 **Phone Setup for Productivity**

### **Focus Modes (iOS) / Do Not Disturb (Android)**

1. **Deep Work Mode** (8 AM - 11 AM, 1 PM - 4 PM)

   - Only emergency contacts
   - Turn off all app notifications
   - Allow only calls and messages from family

2. **Exercise Mode** (6 AM - 7 AM)
   - Allow fitness apps only
   - Music streaming apps
   - No social media or work notifications

### **App Organization**

- **Home Screen**: Only essential apps (Clock, Calendar, Notes, Fitness)
- **Second Screen**: Work apps (Slack, Email, Development tools)
- **Folder**: Social media apps (to reduce impulse checking)

## 🧠 **Browser Setup for Focus**

### **Chrome Extensions**

1. **Cold Turkey Blocker** - Block distracting websites during work hours
2. **Forest** - Browser companion for focus sessions
3. **StayFocusd** - Limit time on specific websites
4. **uBlock Origin** - Remove ads and distractions

### **Bookmarks Organization**

- **Work Folder**: Development resources, documentation
- **Interview Prep**: LeetCode, System Design resources
- **Learning**: Educational content, courses
